About Me

I'm a second-year student in the Computer Security & Investigations (CSI) program at Fleming College in Peterborough, Ontario, working toward a three-year Advanced Diploma graduating in 2027.

My path to cybersecurity started in a lab coat. I completed an Honours BSc in Biomedical Toxicology (Minor in Neuroscience) at the University of Guelph, then spent over a decade working across environmental science, analytical chemistry, toxicological research, and operations management. Every role sharpened the same core skill: finding signal in noise.

That background maps directly onto what security work demands: methodical investigation, precise documentation, and sound judgment under pressure. I'm drawn to the detection and response side of security: understanding how systems are compromised, building detection logic, and constructing evidence-based incident timelines.

Outside of academics, I play drums and classical guitar, and I'm a former competitive rower and coach. I also enjoy exploring food scenes in Peterborough and the Niagara region.

Projects

// Featured · Network Security
Squid Proxy Lab: Access Control & Log Monitoring
Built a fully functional Squid forward proxy in a VMware NAT-mode network. Configured IP whitelisting, user authentication, and access control lists (ACLs) to restrict and log outbound HTTP/HTTPS traffic. Validated behaviour with curl and browser-based tests and set up real-time log ingestion to verify access decisions against defined policy rules.
Squid Proxy VMware Kali Linux Parrot Linux Bash ACL Design
Reflection

Working through ACL logic and troubleshooting authentication failures sharpened my understanding of proxy architecture and traffic filtering. This project demonstrated the direct link between policy configuration and observable log output, which is a skill central to detection engineering and security monitoring roles.

// Scripting & Detection
Wireless IDS/IPS Script (Scapy)
Developed a Python-based wireless intrusion detection script using Scapy. Detected beacon floods, evil twin APs, rogue access points, and deauthentication burst attacks, logging events and triggering configurable response actions.
"Working through real attack signatures at the packet level gave me a much more concrete model of how wireless threats manifest and how defenders can respond programmatically."
Python Scapy Kali Linux 802.11 Wireless Security

AES-GCM Encryption Lab
Implemented AES-GCM symmetric encryption and decryption in a Jupyter Notebook, exploring authenticated encryption, key management, and IV reuse vulnerabilities. Validated outputs and documented the full cryptographic workflow.
"Understanding why authenticated encryption modes exist and what breaks when misused. That understanding built the intuition that's valuable in both security review and incident analysis."

// SIEM & Log Management
Graylog SIEM Lab: Log Ingestion & Alerting
Deployed and configured a Graylog instance to ingest, parse, and search log data from multiple sources. Built structured streams, configured extractors to normalize raw log fields, and set up alert conditions to flag suspicious activity. Tested end-to-end detection workflows by generating events and verifying they triggered the correct alerts.
"Seeing how raw, messy log data gets transformed into searchable, structured events made detection logic feel concrete rather than abstract. This lab gave me a practical understanding of what analysts are actually working with inside a SOC."
Graylog SIEM Log Ingestion Syslog Alerting Linux
